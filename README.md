## Diagnostics

C = Clear canvas
B = box
U = Undo box
F = Flip flop box
R = Render

Q = quit
T = timer show

P = Pause
O = Single step pause

Develeoper keys:

L = activate dev keys

X = Hit boxes
G = Map grid
V = Sprite boxes
I = On/off animations

N = ctx.lineWidth++
M = ctx.lineWidth--


P.S. there are some networking issues. If for example 3 players appear when there are only 2 playing. Please disconnect all players and try again.
